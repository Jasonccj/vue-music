# vue-music

## Build Setup

``` bash
npm install  -->  npm run dev

```

## 概览 项目目录结构

初始化,随项目开发不断完善中

```
.
└── src
    ├── api  空文件夹,待开发
    ├── common 
        ├── fonts 字体,及一些icon基本文件
        ├── image 默认图片文件
        ├── js 相关js文件
        ├── stylus 全局css相关配置文件
    ├── components 自定义组件
    ├── router  路由配置
    ├── store  空文件夹,待开发
```
